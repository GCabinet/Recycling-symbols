# Extension: Recycle Symbols Gallery for LibreOffice

This extension enriches the standard **LibreOffice Gallery** with a comprehensive set of recycling pictograms and environmental signs. It is designed for professional and home use in Writer, Impress, and Draw.

---

## Key Features

* **Vector Quality (SVG):** All symbols are in vector format, ensuring perfect sharpness at any print size.
* **Instant Access:** Symbols are integrated directly into the sidebar; no need to search for external folders.
* **Standardization:** Includes internationally recognized material identification codes.

---

## Gallery Content Overview

| Category | Symbols in the Package |
| :--- | :--- |
| **Universal** | Mobius loop (recycling triangle), Tidyman symbol (person with a bin) |
| **Plastics** | PET (01), HDPE (02), PVC (03), LDPE (04), PP (05), PS (06), O (07) |
| **Paper & Cardboard** | PAP (20, 21, 22) |
| **Metals** | FE (40), ALU (41) |
| **Glass** | GL (70, 71, 72) |

---

## Installation and Usage Guide

### 1. Installation
1. Download the extension file (usually in `.oxt` format).
2. In LibreOffice, go to the **Tools** -> **Extension Manager** menu.
3. Click the **Add** button and select the downloaded file.
4. Restart LibreOffice.

### 2. Inserting Symbols into a Document
1. Open the **Gallery** sidebar (landscape icon on the right sidebar or go to *Insert* -> *Media* -> *Gallery*).
2. Select the **Recycling** (or Eco-Symbols) theme from the list.
3. Simply **drag and drop** the selected symbol onto the document area.

---

> [!TIP]
> **Editing Tip:**
> Since these are "Drawing Objects," you can right-click the symbol in LibreOffice Draw, select **Break**, and then change the colors of individual arrows or backgrounds as needed (e.g., coloring the glass symbol green).

# Rozšíření: Recycle Symbols Gallery pro LibreOffice

Toto rozšíření obohacuje standardní **Galerii LibreOffice** o ucelenou sadu recyklačních piktogramů a environmentálních značek. Je navrženo pro profesionální i domácí použití v aplikacích Writer, Impress a Draw.

---

## Hlavní přednosti

* **Vektorová kvalita (SVG):** Všechny symboly jsou ve vektorovém formátu, což zajišťuje dokonalou ostrost při jakékoli velikosti tisku.
* **Okamžitý přístup:** Symboly jsou integrovány přímo do postranního panelu, není nutné je hledat v externích složkách.
* **Standardizace:** Obsahuje mezinárodně uznávané kódy pro identifikaci materiálů.

---

## Přehled obsahu galerie

| Kategorie | Symboly v balíčku |
| :--- | :--- |
| **Univerzální** | Mobius loop (recyklační trojúhelník), symbol koše (Panáček s košem) |
| **Plasty** | PET (01), HDPE (02), PVC (03), LDPE (04), PP (05), PS (06), O (07) |
| **Papír & Karton** | PAP (20, 21, 22) |
| **Kovy** | FE (40), ALU (41) |
| **Sklo** | GL (70, 71, 72) |

---

## Návod k instalaci a použití

### 1. Instalace
1. Stáhněte soubor rozšíření (formát `.oxt`).
2. V LibreOffice otevřete menu **Nástroje** -> **Správce rozšíření**.
3. Klikněte na tlačítko **Přidat** a vyberte stažený soubor.
4. Restartujte LibreOffice.

### 2. Vložení symbolu do dokumentu
1. Otevřete postranní panel **Galerie** (ikona obrázku v pravé liště nebo menu *Vložit* -> *Médium* -> *Galerie*).
2. V seznamu témat vyberte **Recycling** (nebo Eco-Symbols).
3. Vybraný symbol jednoduše **přetáhněte myší** do plochy dokumentu.

---

> [!TIP]
> **Tip pro úpravy:**
> Jelikož se jedná o objekty typu „Kresba“, můžete po vložení do LibreOffice Draw kliknout na symbol pravým tlačítkem, zvolit **Rozbít** a následně měnit barvy jednotlivých šipek nebo pozadí podle potřeby (např. obarvení na zelenou pro sklo).
